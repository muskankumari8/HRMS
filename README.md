# AI-Powered Human Resource Management System (HRMS)

## Project Overview

An AI-enabled Human Resource Management System developed using Flask, Python, HTML, CSS, JavaScript, SQLite, and Generative AI.

The system provides role-based dashboards and intelligent HR tools for employee management, recruitment, performance analysis, payroll, and workforce analytics.

## AI Features

### Resume Screening AI
Analyzes resumes and ranks candidates based on skills and job requirements.

### Interview Evaluator AI
Provides automated interview assessment and feedback.

### HR Chatbot AI
Answers HR-related questions using Generative AI.

### Attrition Prediction
Predicts employee turnover risk using machine learning.
